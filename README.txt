-----------------
Friend-Pizzeria

ระบบขายพิซซ่าออนไลน์ สามาถรสร้างสรรค์พิซซ่าได้เองด้วยตัวคุณ

ข้อกำหนดเบื้องต้น
-----------------
ก่อนเริ่มต้น โปรดตรวจสอบให้แน่ใจว่าคุณได้ติดตั้งสิ่งต่อไปนี้แล้ว:

- Node.js (เวอร์ชัน X.X.X ขึ้นไป) : https://nodejs.org/
- npm (มาพร้อมกับ Node.js) : https://www.npmjs.com/
- Git (สำหรับโคลนรีโพสิทอรี) : https://git-scm.com/

การติดตั้ง
-----------
1. โคลนโปรเจ็ค:

   git clone https://github.com/nuttawit007/Friend-Pizzeria.git
   cd your-repo

2. เริ่มเซิร์ฟเวอร์สำหรับพัฒนา:

   - สำหรับ Linux/macOS:
     ./start.sh

   - สำหรับ Windows:
     start.bat

3. เปิดเว็บเบราว์เซอร์ไปที่:
   http://localhost:3000
